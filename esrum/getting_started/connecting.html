Configuring MobaXterm¶

For Linux users¶

Linux users can connect to the cluster using the following command, replacing abc123 with your KU username:

$ ssh abc123@esrumhead01fl.unicph.domain

It is recommended to add an entry for the cluster to your .ssh/config file:

$ cat ~/.ssh/config
Host esrum esrumhead01fl.unicph.domain
    HostName esrumhead01fl.unicph.domain
    User abc123

This allows you connect to the server using the abbreviated name esrum and without having to specify your username:

$ ssh esrum

Note

Note that the cluster does not support authentication through a public SSH key and that you have to enter your password when connecting to the server.
